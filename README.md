# Serverless toy project
Scrapping [KoreanPost](http://www.nzkoreapost.com/bbs/board.php?bo_table=market_recruit) Recuruit page 

# Built with
* AWS Lambda
* AWS API Gateway
* AWS RDS(MySQL)
* AWS CloudWatch
* Chalice framework (Python)
